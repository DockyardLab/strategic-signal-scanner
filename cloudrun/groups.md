# Cloud Run Source Groups

This repo keeps a few source groups so you can choose how broad or focused the daily report should be.

The public default. Stable, broad, and good for a daily digest.

- Sequoia Capital
- Stratechery
- Follow Builders

## `fast`

Fast-moving AI and infra signal. Good when you want a broader and quicker read on the field.

## `upstream`

Upstream builder signal from the follow-builders line. Useful for builder-first monitoring.

## `front` and `youtube_front`

Top-row sources and featured YouTube/public channels.

## `podcasts_rss` and `podcasts_web`

Podcast-centric sources. Use these if you want more long-form interviews and conversations.

## `slow`

Research-heavy, slower-moving sources.

## `late`

Long-form, authorial, and opinion-rich sources.
